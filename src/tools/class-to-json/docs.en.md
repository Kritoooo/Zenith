# Class to JSON

Generate JSON templates and mock data from Java / Go / Python class definitions.

## Features
- Recognizes primitives, arrays, and common collections (List/Map)
- Go structs with json tags
- Outputs JSON Template and Mock Data
- Adjustable mock rows (1-10)
- One-click copy

## Steps
1. Choose language (Java / Go / Python).
2. Paste class/struct definitions.
3. Set Mock rows and click Generate.
4. Copy template or mock data from the right.

## Notes
- Unknown types render as null or empty objects.
- If no fields are parsed, check class format or comment placement.
