# YT-DLP

Generate yt-dlp download commands (video/audio).

## Features
- Video/audio mode
- Format presets and custom format
- Merge format, subtitles, thumbnails
- Output template/folder, rate limit, cookies, download archive
- Extra args and one-click copy

## Steps
1. Paste a target URL (video/playlist/channel).
2. Configure format, subtitles, output options.
3. Copy the command and run locally.

## Notes
- This tool only generates commands; it does not download.
- FFmpeg is required for merge/embedded subtitles/thumbnails.
