# JSON Formatter

将 JSON 格式化为可读的树状或原始文本，并支持压缩输出。

## 功能
- 格式化：2/4 空格缩进
- 压缩：输出为单行
- Tree / Raw 视图切换，展示体积与节点数
- 字号可调
- 展开 / 折叠所有节点
- 大体积防护：超大文本自动切 Raw，可按需强制树形
- 一键复制输出内容

## 使用步骤
1. 在 Input 粘贴 JSON。
2. 点击 Format 或 Minify。
 3. 在 Output 里切换 Tree 或 Raw，并按需调整字号。
 4. 可折叠输入框，专注输出。
 5. 需要复制时点击 Copy。

## 注意
- 仅支持标准 JSON（不支持注释或尾逗号）。
- 若浏览器阻止剪贴板，请切换到 Raw 模式手动复制。
- 体积超过约 120 KB 或节点多于约 8,000 时，树形视图会被暂停，可改用 Raw 或点击强制树形。
