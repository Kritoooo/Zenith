# Color Converter

Convert quickly between Hex / RGB / HSL.

## Features
- Hex, RGB, HSL in sync
- Color picker with live preview
- Presets and random colors
- One-click copy for each format

## Steps
1. Enter a color value or use the picker.
2. Other formats sync automatically.
3. Click Copy for the format you need.

## Notes
- Hex supports 3 or 6 digits (# optional).
- RGB range 0–255; HSL S/L range 0–100.
