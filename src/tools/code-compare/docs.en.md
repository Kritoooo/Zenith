# Code Compare

Compare two text blocks with highlights and image export.

## Features
- Split / Inline views
- Line diffs with word/character highlights
- Added/removed/changed line stats
- Export comparison as PNG

## Steps
1. Enter Original and Updated.
2. Click Compare.
3. Switch Split/Inline and Realtime diff as needed.
4. Click Export PNG to share.

## Notes
- Realtime diff provides finer (character-level) highlights but can be slower for long text.
- Exported images preserve the current theme and view mode.
