# Image Compress

Compress images in the browser and download the result.

## Features
- Drag/drop or upload JPG, PNG, WebP
- Select output format and quality
- Optional max width/height auto-resize
- Quick presets (Crisp / Balanced / Small)
- Before/after preview and download

## Steps
1. Upload or drag an image.
2. Adjust format, quality, or size limits.
3. Click Compress to generate output.
4. Click Download to save.

## Notes
- PNG is lossless; the quality slider is disabled.
- Compression runs locally; images are not uploaded.
