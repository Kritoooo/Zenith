# WebGPU OCR

Extract text from images in the browser with PaddleOCR ONNX.

## Features
- Multi-language recognition (CN/EN/KR/TH, etc.)
- WebGPU-accelerated, fully on-device
- Text + confidence stats
- Bounding boxes (toggle)
- Low-confidence filter and threshold
- One-click copy

## Steps
1. Upload or drag an image.
2. Select language and click Recognize.
3. Review results, confidence, and boxes.
4. Click Copy text to copy.

## Notes
- Requires a WebGPU-capable browser (Chromium recommended).
- First run downloads and caches models.
- Confidence is a heuristic; boxes are approximate.
