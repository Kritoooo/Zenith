# WebGPU OCR

在浏览器内使用 PaddleOCR ONNX 提取图片文字。

## 功能
- 多语言识别（含中/英/韩/泰等）
- WebGPU 加速、纯前端推理
- 识别结果与置信度统计
- 框选文字位置（可开关）
- 低置信度筛选与阈值调节
- 一键复制识别文本

## 使用步骤
1. 上传或拖拽图片。
2. 选择语言并点击 Recognize。
3. 查看结果、置信度与框选。
4. 点击 Copy text 复制内容。

## 注意
- 需要支持 WebGPU 的浏览器（推荐 Chromium 内核）。
- 首次使用会下载模型并缓存。
- 置信度仅为参考，识别框为近似位置。
