# Base64

Quickly convert between Base64 and plain text.

## Features
- Encode / Decode modes
- Swap uses output as the next input
- One-click copy
- Ignores whitespace when decoding

## Steps
1. Paste text or Base64 into Input.
2. Click Encode or Decode.
3. Click Swap to reuse output.
4. Click Copy to copy the result.

## Notes
- Decoding requires valid Base64 input.
- Clipboard access depends on browser permissions.
