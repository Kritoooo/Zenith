# Address Generator

基于 OpenStreetMap 生成包含姓名、电话与地址的示例资料。

## 功能
- 多国家/地区选择
- 生成姓名、性别、电话与完整地址
- 单项复制 / 一键复制全部
- 地图预览与跳转到 OpenStreetMap
- 保存列表与备注（本地存储）

## 使用步骤
1. 选择 Country，点击 Generate。
2. 点击任意字段即可复制，或使用 Copy all。
3. 需要保留时填写备注并点击 Save。
4. 通过地图预览或 Open in OpenStreetMap 查看位置。

## 注意
- 地址来自 OpenStreetMap Nominatim 反向地理编码，可能需要多次尝试。
- 生成结果仅用于测试/演示，请勿作为真实身份信息使用。
- 已保存列表仅存于浏览器本地。
