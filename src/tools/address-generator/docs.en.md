# Address Generator

Generate sample profiles with name, phone, and address using OpenStreetMap.

## Features
- Multi-country/region selection
- Generates name, gender, phone, and full address
- Copy single field / copy all
- Map preview and open in OpenStreetMap
- Saved list with notes (local storage)

## Steps
1. Choose Country and click Generate.
2. Click any field to copy, or use Copy all.
3. Add a note and click Save if you want to keep it.
4. View the location in map preview or Open in OpenStreetMap.

## Notes
- Uses OpenStreetMap Nominatim reverse geocoding; it may require retries.
- For testing/demo only; do not use as real identity data.
- Saved list is stored locally in the browser.
